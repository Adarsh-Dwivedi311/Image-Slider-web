# Image-Slider-web
An interactive and visually appealing Image Slider Web Application built using modern web technologies. This project demonstrates smooth transitions, responsive design, and user-friendly navigation for showcasing images dynamically.
_________________________________________________________________________
✨ Features
🎯 Smooth sliding animations
🖼️ Dynamic image transitions
📱 Fully responsive design (Mobile + Desktop)
⚡ Fast and lightweight
🖱️ Click-based navigation (thumbnails support)
🎨 Clean and modern UI
____________________________________________________________________________
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & animations
JavaScript (ES6) – Functionality
____________________________________________________________________________
📂 Project Structure
📁 Image-Slider
 ┣ 📄 index.html
 ┣ 📄 style.css
 ┣ 📄 script.js
 ┗ 📁 images
___________________________________________________________________________
 ⚙️ How It Works
Thumbnails are used to select images
JavaScript updates the slider position using transform: translateY()
Smooth transitions create a seamless sliding effect
